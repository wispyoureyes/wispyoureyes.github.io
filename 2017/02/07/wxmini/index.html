微信小程序踩坑

发表于 2017-02-06

###微信小程序在电脑能获取到数据，但是手机上却无法显示数据

  • 微信小程序请求接口时只能使用https协议，电脑上的微信开发者工具不检测域名，所以可以显示

    ios上可以正常显示数据，但是在安卓设备上却无法显示数据

  • 数据接口url必须加上.json后缀否则在安卓设备上会直接获取到页面的代码而不是json数据

    如何调用page方法中的变量

this.data.reportId

使用onReachBottom实现上拉加载后当页面没有数据时也会出现加载中的动画

  • 可以在onReachBottom中套用if条件，判断当列表页的数据量大于一定数量时才执行加载更多的方法，当页面没有数据时就不会出现加载中的动画了
