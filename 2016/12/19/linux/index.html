linux学习笔记

发表于 2016-12-18

命令提示符

其中：
root:当前登录用户
localhost:主机名
~:当前所在目录，此处为家目录

#:root超级用户的提示符，如果是普通用户，则为$

创建目录mkdir

  • mkdir [目录名] 直接在当前目录下创建目录

  • mkdir -p [目录名] 可以在当前目录下创建带有子目录的目录如：

    mkdir -p [目录名/子目录名]

切换所在目录cd

  • cd ~:进入当前目录的家目录
  • cd-:进入上次目录
  • cd..:进入上一级目录
  • cd:回到家目录

查询所在目录位置：pwd

删除空目录：rmdir

删除文件或目录：rm

rm -rf [文件或目录]强制删除

r 表示可以同时删除文件和目录，f表示强制删除

  • rm [文件或目录]——只可以删除文件，删除时提示是否确认删除
  • rm -r [文件或目录]——可以删除文件也可以删除目录，删除时提示是否确认删除
  • rm -rf [文件或目录]——将不做任何提示删除文件或目录

创建文件 touch

  • $ touch adsfasd.txt wergjlkas.txt
    ——使用 touch 命令创建 2 个文件，后缀都为 txt
  • 一次性创建多个文件：$ touch love_{1..10}_linux.txt

获取并打印文件 cat [文件路径]

  • $ cat /etc/group | sort
  • | sort表示将读取的文本进行字典排序再输出

    基本概念

    终端和控制台的区别

  • 终端是附加设备，同过串口链接到计算机上（现在的linux系统上都用虚拟终端无需用串口链接），而控制台式计算机的基本设备，本身就有
  • 一天计算机可以有多个终端但是只能有一个控制台
  • 控制台也有和终端一样的功能，计算机操作系统中，与终端不相关的信息，比如内核消息，后台服务消息，都可以显示到控制台上，但不会显示到终端上。
  • 能直接显示系统消息的那个终端称为控制台，其他的则称为终端。
  • 现在的linux系统已经弱化了终端和控制器的区别，但是其他系统中却很明显的有虚拟终端和控制台的区别

    小技巧，快捷键

  • tab 补全命令
  • control + c 强行终止当前程序
  • Ctrl+d 键盘输入结束或退出终端
  • Ctrl+s
  • 暂停当前程序，暂停后按下任意键恢复运行
  • Ctrl+z 将当前程序放到后台运行，恢复到前台为命令fg
  • Ctrl+a 将光标移至输入行头，相当于Home键
  • Ctrl+e 将光标移至输入行末，相当于End键
  • Ctrl+k 删除从光标所在位置到行末
  • Alt+Backspace 向前删除一个单词
  • Shift+PgUp 将终端显示向上滚动
  • Shift+PgDn 将终端显示向下滚动

通配符

-通配符是一种特殊语句，主要有星号（*）和问号（?），用来对对字符串进行模糊匹配（比如文件名，参数名）。通配符由shell来搜索匹配，并补全命令行，然后才由命令行处理

  • $ ls *.txt可以模糊搜索
    • 匹配 0 或多个字符
  • ? 匹配任意一个字符
  • [list] 匹配 list 中的任意单一字符
  • [!list] 匹配 除list中的任意单一字符以外的字符
  • [c1-c2] 匹配 c1-c2 中的任意单一字符如：[0-9] [a-z]
  • {string1,string2,…} 匹配 sring1 或 string2 (或更多)其一字符串
  • {c1..c2} 匹配 c1-c2 中全部字符 如{1..10}

sudo命令

  • Sudo是Unix/Linux平台上的一个非常有用的工具，它允许系统管理员分配给普通用户一些合理的“权利”，让他们执行一些只有超级用户或其他 特许用户才能完成的任务
  • 比如：运行一些像mount，halt，su之类的命令，或者编辑一些系统配置文件，像/etc/mtab，/etc /samba/smb.conf等。这样以来，就不仅减少了root用户的登陆次数和管理时间，也提高了系统安全性。
  • root权限就是超级管理员权限，最高用户等级

用户管理

  • 创建用户 sudo adduser [用户名]
  • 切换用户 su -l [用户名]
  • 查看用户组 groups [用户名]
  • 更改用户组 sudo usermod -G sudo lilei
  • 删除用户 sudo deluser lilei –remove-home
